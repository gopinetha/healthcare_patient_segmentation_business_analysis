## Overview
This project focuses on analyzing patient visit behavior in a healthcare setting to improve operational efficiency and resource planning.

## Business Problem
Healthcare organizations lack visibility into patient visit patterns, leading to inefficient staffing and increased wait times.

## Objective

- Data visualization
- Business requirements documentation

## Key Insights
- Identified high-frequency patient clusters
- Detected anomalous visit behaviors
- Highlighted operational planning opportunities
